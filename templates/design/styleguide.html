Design system

Core tokens and components. Build new features from this page only.

Rules

  • Compose screens from sections on this page (base.css only).
  • Page-specific UI uses scoped area CSS (e.g. workout-sets-table) — not listed here.
  • New core classes: add to base.css and this page first.
  • Promote to base.css when a second feature needs the same pattern.
  • Mobile-first: default styles target a phone viewport.
